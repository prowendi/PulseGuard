新建 Bot

向 @BotFather 创建后获得 Token，复制到下方即可。

Telegram 全双向（push + listener + 内联按钮）；Lark 支持 Webhook（仅 push）与 应用机器人（双向 + 事件订阅）；SMTP 仅推送邮件，无命令支持。

Webhook：单向群推送；App：OAuth + IM API + 事件订阅，可双向交互。

Telegram 填 数字:字母；Lark Webhook 填完整 URL（含 https://open.feishu.cn/open-apis/bot/v2/hook/…）。

事件订阅 URL 请填 POST /api/v1/lark/events。Encrypt Key 用于校验 X-Lark-Signature。

Channel.chat_id 在 SMTP 下当作收件人，多人用逗号分隔。模板渲染后的第一行会作为邮件主题，\n\n 之后是正文。parse_mode=HTML 会以 text/html 发送。

编辑 Bot

修改 token 或 platform 会自动重启 listener；改名称/描述则不重启。

留空保留现有 token；填新值会触发 listener 重启（Lark 平台无 listener，仅替换 token）。
