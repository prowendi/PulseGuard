新建 Bot

向 @BotFather 创建后获得 Token，复制到下方即可。

Telegram 全双向（push + listener + 内联按钮）；Lark 仅 push（custom bot webhook，无 listener / 无按钮 / 无 edit）。

Telegram 填 数字:字母；Lark 填完整 webhook URL（含 https://open.feishu.cn/open-apis/bot/v2/hook/…）。

编辑 Bot

修改 token 或 platform 会自动重启 listener；改名称/描述则不重启。

留空保留现有 token；填新值会触发 listener 重启（Lark 平台无 listener，仅替换 token）。
